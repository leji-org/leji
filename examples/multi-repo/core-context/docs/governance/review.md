# Review

- Anyone, person or agent, proposes a change to this context layer as an ordinary pull request.
- A person on the owning team approves before it becomes canonical; agents never self-approve.
- A change that alters meaning appends a `docs/context-changelog.json` entry in the same pull request.
- Mounted sibling layers are read here, not edited; each sibling's own owner approves its changes.
