# Glossary

- **Account**: the organization-level record a customer holds with Acme; every product scopes its data to an account.
- **Customer**: a person or company that holds one or more accounts.
- **Money**: an integer amount in minor units paired with an ISO 4217 currency, never a bare float.
- **Environment**: one of `dev`, `staging`, or `prod`; identifiers are never shared across environments.
